# 🐼 PandaBook — персональный диетолог

Заполни холодильник → получи сбалансированный рацион на день с учётом цели
(похудеть/поддерживать/набрать), сроков годности продуктов и диет-ограничений.
Плюс умный список докупок и рецепты от ИИ.

## Стек
- **FastAPI** — ядро и вся логика (`/api/*`), запускается через uvicorn.
- **Flask** — обязательный слой, примонтирован в FastAPI через `a2wsgi`, отдаёт страницы.
- **SQLAlchemy + SQLite** — хранилище.
- **Собственный ИИ PandaBook** — рецепты, объяснения рациона и список докупок.

## Запуск
```bash
python -m venv .venv && .venv\Scripts\activate   # Windows
pip install -r requirements.txt
uvicorn app.main:app --reload
```
Открой http://127.0.0.1:8000 — страница Flask с кнопкой демо-рациона.
Демо-API: `GET /api/diet/demo`. Полный флоу: `POST /api/diet/plan`.

Тесты: `pytest`

## Два режима (`POST /api/diet/plan`, поле `mode`)
- **`fast`** (бесплатный) — только алгоритмический движок: мгновенный рацион +
  список докупок. Без ИИ, детерминированно, офлайн.
- **`thinking`** (Pro) — тот же план + ИИ PandaBook: объяснение
  рациона под цель и рецепт из продуктов дня. Учитывает свободный запрос
  пользователя (поле `request`, напр. «хочу побольше белка»).

## Что реализовано (Backend-1)
- Расчёт КБЖУ-целей (Mifflin-St Jeor → TDEE → цель → макросы).
- Генератор рациона: жадное распределение по приёмам пищи, **приоритет
  истекающих продуктов** (анти-фудвейст), фильтры (веган/вег/халяль/без
  глютена/без лактозы/аллергены).
- Рекомендатель докупок по дефициту макросов.
- ИИ-слой PandaBook: рецепты, объяснения и докупки; два режима fast/thinking.
- Дашборд прогресса + панда-геймификация (история КБЖУ, стрики, настроение панды).
- Демо-эндпоинт и тесты движка (10/10).
